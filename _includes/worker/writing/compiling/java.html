IronWorker runs your Java programs from JAR files, so you're going to need to compile that Hello.java file and create a JAR out of it.

First, you'll need to create a manifest.txt file in the root of your project (~/iron/myworker in our example). It should look like this:

Main-Class: Hello

Next, you'll need to run the following commands from the root of your project (~/iron/myworker in our example):

{% highlight bash %} $ javac Hello.java $ jar cfm hello.jar manifest.txt {% endhighlight %}

This will create hello.jar file in the root of your project.
